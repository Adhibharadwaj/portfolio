# public assets

Files placed here are served from the site root (respecting the Vite `base` path)
and are bundled as-is during `npm run build`.

## Resume

Save your CV as **`resume.pdf`** in this folder:

```
public/resume.pdf
```

The "Resume" button in the top-right of the site links to it automatically.
After deploying, it is available at `/portfolio/resume.pdf`.
